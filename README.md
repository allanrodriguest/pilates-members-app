# Pilates Members App

> Content:

- Create, Read, Update and Delete members
- Simple API
- Express Router
- Static Files
- Template Engine Setup with Views
- Middleware
- Nodemon
- Bootstrap

## Quick Start

```bash
# Clone this repo

$ git clone https://github.com/allanrodriguest/pilates-members-app.git

# Install dependencies:

yarn or npm install

# Serve on localhost:8000

npm run dev
```
